**Install dependencies with homebrew**

```
brew install jq
brew install mapbox/cli/mapbox
brew install jp2a
```

**Clone repo**

```
git clone https:/github.com/maning/prosm
cd prosm
```

**Run the app**

Provide your Mapbox access token

```
mapbox --access-token MY_TOKEN ...
```

Get your story

```
sh prosm.sh <username>
```
